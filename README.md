# Just Music — OLED 风格本地音乐播放器

一款基于 PyQt6 开发的 Windows 本地音乐播放器，采用纯黑 OLED 风格界面设计，支持多种音频格式与内嵌/外挂歌词，追求极简视觉与纯粹播放体验。

---

## ✨ 核心特性

- **OLED 纯黑界面**：全 `#000000` 背景，低亮度护眼，专为 OLED 屏幕优化
- **无边框圆角悬浮窗**：380×680 固定尺寸，16px 圆角，支持置顶与自由拖动，适合作为桌面音乐小部件
- **多格式支持**：MP3、FLAC、WAV、M4A、OGG（自动递归扫描文件夹）
- **元数据自动解析**：通过 mutagen 读取标题、艺术家、专辑、年份、内嵌封面与歌词
- **圆角封面展示**：自定义 `RoundedLabel` 组件，带抗锯齿圆角边框，专辑封面 64×64 展示
- **动画歌词切换**：支持标准 LRC 格式，当前行与下一句双行显示，切换时带平滑滚动过渡
- **非线性音量控制**：采用平方曲线映射，更符合人耳响度感知
- **记忆用户文件夹**：自动记住上次选择的音乐目录，下次启动直接加载
- **矢量图标控件**：播放/暂停/切歌/文件夹/窗口控制均采用 `QPainter` 自绘，无位图资源依赖
- **内置字体回退**：优先使用更纱黑体（SarasaUiSC）渲染界面，缺失时自动回退至系统默认微软雅黑
- **智能封面回退**：内嵌封面 → 同目录同名 `.jpg` → `cover.jpg`

- **信息岛**：64×64 圆角封面、歌名、年份、艺术家与专辑信息
- **控制岛**：音量滑块、矢量播放/暂停/切歌按钮、可拖动进度条（支持预览）、时间显示
- **歌词岛**：双行歌词视图，当前行高亮，切换时自动滚动过渡
- **赏析岛**：显示 FLAC `description` 标签或自定义文本，带滚动区域
- **列表岛**：可滚动歌曲列表，当前播放项高亮，悬停反馈

## 📦 依赖要求

如果你想从源码运行或二次开发，需要安装以下依赖：

```
PyQt6>=6.4
mutagen>=1.46
```

> 若仅需播放 WAV/MP3 而不读取元数据，可跳过 `mutagen` 安装，程序将自动降级运行。

## 🚀 从源码运行

### 1. 安装依赖

```bash
pip install PyQt6 mutagen
```

### 2. 准备字体（可选）

程序内置字体回退机制：优先使用更纱黑体（SarasaUiSC），若打包时未内嵌或运行环境缺失，则自动回退至系统默认的 `Microsoft YaHei`。

源码运行时，可将以下字体文件置于源码同级目录，打包工具会自动将其内嵌：

- `SarasaUiSC-Regular.ttf`
- `SarasaUiSC-Bold.ttf`

### 3. 运行

```bash
python main.py
```

程序启动后会优先加载上次使用的音乐文件夹（通过 `QSettings` 持久化）；若首次运行，则尝试加载默认路径：
```
C:\Users\你的用户名\Music\Just Music
```
也可通过顶部标题栏的文件夹图标手动指定音乐目录。

## 🎵 文件组织建议

为获得最佳体验，建议按以下方式整理曲库：

```
Just Music/
├── 春日影.mp3
├── 春日影.lrc          ← 外挂歌词（可选）
├── 春日影.jpg          ← 同名封面（可选）
├── 迷星叫.flac         ← FLAC 内嵌封面/歌词优先读取
└── cover.jpg           ← 文件夹通用封面（回退）
```

## 🛠️ 技术细节

| 模块 | 说明 |
|------|------|
| `QMediaPlayer` + `QAudioOutput` | Qt6 原生多媒体后端，无需额外解码器 |
| `mutagen` | 音频元数据解析，支持 ID3 / Vorbis Comment / APE |
| `LRCParser` | 正则解析 `[mm:ss.xx]` 时间戳，支持多时间标签单行 |
| `RoundedLabel` | 自定义 `QLabel`，`QPainter` + `QPainterPath` 实现圆角裁切与边框 |
| `IconButton` | 自绘矢量图标按钮，支持 play/pause/prev/next/folder/close/minus 状态 |
| `LyricsView` | 双行歌词容器，基于 `QPropertyAnimation` 实现 250ms 平滑滚动切换 |
| `Island` | 卡片式容器，统一圆角、背景色与边框，构建模块化布局 |
| `DraggableMixin` | 无边框窗口拖动支持，通过 `mousePressEvent` / `mouseMoveEvent` 实现 |
| `QSettings` | 持久化用户最后选择的音乐文件夹路径 |
| `resource_path` | 兼容 PyInstaller `--onefile` 打包后的资源路径解析 |
| 进度预览 | `sliderMoved` 时计算预览毫秒数，实时驱动歌词更新，释放后跳转 |

## 📋 自行打包为可执行文件

如果你想自己打包，使用 PyInstaller：

```bash

```

> 若使用 FLAC/OGG 等格式，请确保目标机器已安装相应 DirectShow 滤镜或 Qt 多媒体插件。
